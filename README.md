# Auto-Tracker-legal-management-system
Auto Tracker is a modern legal practice management system designed to streamline time tracking, billing, client management, and trust accounting in one unified platform.
Built as a lightweight alternative to traditional legal software, it focuses on **real-time activity capture, automation, and data-driven insights** to improve billing accuracy and productivity.
Features
⏱️ Time Tracking:** Live timer with 6-minute billing increments and quick manual entry
⚡ Auto-Detection (Simulated):** Captures emails, meetings, document work, and calls
🧾 Invoicing:** Generate and export invoices from approved time entries
👥 Clients & Matters:** Manage clients and link them to billable matters
🏦 Trust Accounting:** Track deposits, withdrawals, and client balances
📊 Insights:** Visualize time allocation, utilization, and efficiency metrics
📅 Reports:** Monthly summaries with Excel/PDF export

🧠 Problem It Solves
Legal professionals often rely on manual time tracking, leading to lost billable hours, inefficiencies, and fragmented workflows.
Auto tracker addresses this by:
Automating activity capture
Reducing manual input
Providing real-time visibility into productivity and billing
🏗️ Tech Stack
HTML, Tailwind CSS, Vanilla JavaScript
ApexCharts (analytics)
Lucide Icons
SheetJS (Excel export)
Single Page Application (SPA) architecture
⚙️ Status
⚠️ Prototype / demo application with simulated features. Backend integration and real-world API connections are planned.
💡 Vision

To build a next-generation legal operating system that automatically captures work, improves billing accuracy, and gives firms complete visibility into their operations.
